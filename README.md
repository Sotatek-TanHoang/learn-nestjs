## Description

[Nest](https://github.com/nestjs/nest) Monorepo starter kits for application with kafka, redis, postgres

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start @app/<app-name>

# watch mode
$ npm run start:dev @app/<app-name>

# production mode
$ node dist/apps/<app-name>/main.js
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
